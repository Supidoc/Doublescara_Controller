Executes a Cartesian move through the motor module and waits for completion.

This helper updates kinematics commanded state, executes a synchronized motor move, and refreshes the actual pose from motor feedback.

Parameters

[in]rotateZetaWhether to rotate the end-effector by deltaZetaE.
[in]deadlineDeadline for all queued operations.
Returns
kStatus_Success if the full sequence succeeds.
