status_t SYW_cmd_check_all ( CHD_CmdHandle_t * cmdHandles,
size_t count,
EventBits_t * outResults )

Check whether all commands have completed, without waiting.

Polls all command handles once and returns immediately.

Parameters
[in]cmdHandlesArray of command handles to check.
[in]countNumber of handles in the array.
[out]outResultsOptional array to receive result bits (CHD_CMD_BIT_SUCCESS, CHD_CMD_BIT_FAILURE, CHD_CMD_BIT_TIMEOUT) for each command. Caller must allocate array of size count. Can be NULL.
Returns
kStatus_Success if all commands have CHD_CMD_BIT_SUCCESS bit set. kStatus_Fail if any completed command has CHD_CMD_BIT_FAILURE bit set. kStatus_Timeout if any command is still pending, or any has CHD_CMD_BIT_TIMEOUT.

size_t count,
TickType_t deadline,
EventBits_t * outResults )

Wait for all commands to complete.

Blocks until all commands complete or the deadline is reached.

Parameters
[in]cmdHandlesArray of command handles to wait for.
[in]countNumber of handles in the array.
[in]deadlineDeadline for waiting (in ticks).
[out]outResultsOptional array to receive result bits (CHD_CMD_BIT_SUCCESS, CHD_CMD_BIT_FAILURE, CHD_CMD_BIT_TIMEOUT) for each command. Caller must allocate array of size count. Can be NULL.
Returns
kStatus_Success if all commands have CHD_CMD_BIT_SUCCESS bit set. kStatus_Fail if any command has CHD_CMD_BIT_FAILURE bit set, deadline exceeded, or not all commands completed. kStatus_Timeout if any command has CHD_CMD_BIT_TIMEOUT bit set.
