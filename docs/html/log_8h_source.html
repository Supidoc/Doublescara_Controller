2 * @file log.h
3 * @brief Module for logging data to an SD card.
4 *
5 * This module provides functionality for logging messages to an SD card with
6 * optimized string assembly using snprintf(). It supports multiple log levels (DEBUG, INFO,
7 * WARN, ERROR), task-safe message queuing, and session-based log file management.
8 * Log output format: [LEVEL]:[TIMESTAMP]:[MESSAGE]\r\n
9 * The log files can be retrieved using CLI commands.
10 *
11 * @note Ensure the disk module is initialized before using this module.
