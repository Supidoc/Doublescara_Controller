void ESTOP_ClearEstop ( void )

Clear the emergency stop.

Clears the global E-Stop flag and allows normal operation to resume. Callers should ensure it is safe to resume motors and actuators before calling this function.

◆ ESTOP_SetEstop()

void ESTOP_SetEstop ( void )

Assert the emergency stop.

Sets the global E-Stop flag and triggers any immediate shutdown actions required by the platform (motor disable, brake engage, etc.). This function is safe to call from task context. If ISR safety is required a separate API should be provided by the implementation.

bool estopFlag
extern

Global emergency-stop flag.

True when an emergency-stop is active. Modules that perform motion or drive external actuators should check this flag and refrain from issuing commands while it is set.
