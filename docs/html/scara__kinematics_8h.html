typedef enum _SK_Configuration SK_Configuration_t

SCARA elbow configuration options.

Specifies which elbow configuration to use for inverse kinematics. Different configurations are used to resolve the elbow-up or elbow-down solution ambiguity in SCARA arm kinematics.
