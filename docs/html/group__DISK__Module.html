◆ DISK_init()

status_t DISK_init ( void )

Initializes the disk and mounts the filesystem.

This function mounts the filesystem on the specified disk path. It must be called before performing any file operations on the disk. This should be one of the first initialization functions called in your application startup sequence.

Note
This function is NOT task-safe and must be called during system initialization.
Other modules that depend on disk access (e.g., LOG) require this to be initialized first.
Warning
File operations are not safe until this function completes successfully.
Note
Ensure SD card hardware is properly connected and detected before calling this.
Returns
kStatus_Success if the filesystem is successfully mounted. kStatus_Fail if the mounting operation fails (card not detected or corrupted).
