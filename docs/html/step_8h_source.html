1/************************************************************
2 * @file step.h
3 * @brief Module for controlling stepper motors with FTM
4 *
5 * This module provides functionality for controlling stepper motors using
6 * the FTM peripheral. It supports acceleration
7 * and deceleration profiles with trapezoidal velocity profiles for smooth
8 * motor control. The module uses FreeRTOS queues for task-safe movement
9 * commands.
10 *
11 * @note Ensure the FTM peripherals and GPIO are initialized before using this module.
12 * @author dg
13 * @date 4 Jan 2026
14 ************************************************************/
