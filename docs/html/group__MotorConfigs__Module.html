Detailed Description

Factory functions for motor configuration structures.

Provides five factory functions for complete motor system: SCARA System (UART1):

  • M_L_Arm(): Left SCARA arm motor (serial address 0, FTM3 ch.2, GPIOC pin 11)
  • M_R_Arm(): Right SCARA arm motor (serial address 1, FTM3 ch.1, GPIOD pin 2)
  • M_Platform(): Platform motor (serial address 2, FTM3 ch.0, GPIOD pin 0)
    Auxiliary System (UART1):
  • M_Magnet(): Electromagnet control (serial address 3, FTM3 ch.3, GPIOC pin 10, 7.5° steps)
    Independent System (UART0):
  • M_Rotation(): Rotation/indexing motor (serial address 0, FTM3 ch.4, GPIOC pin 9)
    All motors controlled via PCA9555A I2C IO expander for direction pins.
    SCARA arms: 1.8° step, 16x microstep, 3.0x reduction, 0.8A run, 0.1A hold.
    Platform: 1.8° step, 1x microstep, 1.0x reduction, 0.1A both.
    Magnet: 7.5° step, 1x microstep, 1.0x reduction, 0.1A both.
    Rotation: 1.8° step, 16x microstep, 1.0x reduction, 0.2A run, 0.1A hold.

MTR_MotorConfig_t M_Magnet ( void )

Creates and returns the configuration for the magnet motor.

Initializes all stepper and TMC2209 driver parameters for the electromagnet control motor. Configured with 7.5° step angle for reduced step resolution, 1x microstepping. Uses UART1 serial address 3. Hardware: FTM3 channel 3, GPIO C pin 10, PCA9555A port bit 3 for direction. Acceleration: 3200°/s², velocity: 5760°/s. Hold and run currents: 0.1A.

MTR_MotorConfig_t M_Rotation ( void )

Creates and returns the configuration for the rotation motor.

Initializes all stepper and TMC2209 driver parameters for the mechanical rotation/indexing motor. Uses 1.8° step angle with 16x microstepping for fine position control. Configured on UART0 (serial address 0) separate from the SCARA arm system. Hardware: FTM3 channel 4, GPIO C pin 9, PCA9555A port bit 4 for direction. Acceleration: 1600°/s², velocity: 2880°/s. Hold current: 0.1A, run current: 0.2A.
