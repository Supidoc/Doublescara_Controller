status_t MTT_init ( void )

Initializes the motor test module.

Sets up the motor test module for operation. This function should be called once during system initialization before using any other motor test functions.

Returns
Status code indicating success or failure
  • kStatus_Success: Module initialized successfully
  • Other: An error occurred during initialization

◆ MTT_task()

void MTT_task ( void * pvParameters)

FreeRTOS task for motor testing and diagnostics.

This is the main task function for the motor test module. It initializes the stepper motor driver, configures the TMC2209 stepper motor driver, and registers CLI commands for motor control. This task runs indefinitely and should be created as a FreeRTOS task.

Parameters
[in]pvParametersPointer to task parameters (typically NULL for this task)
Note
This function should be registered as a FreeRTOS task and will block indefinitely, yielding periodically.
