\doxysubsection{Detailed Description}
MCG mode change configuration structure. 

When porting to a new board, set the following members according to the board setting:\+
\begin{DoxyEnumerate}
\item frdiv:\+ If the FLL uses the external reference clock, set this value to ensure that the external reference clock divided by frdiv is in the 31.\+25 k\+Hz to 39.\+0625 k\+Hz range.
\item The PLL reference clock divider PRDIV:\+ PLL reference clock frequency after PRDIV should be in the FSL\+\_\+\+FEATURE\+\_\+\+MCG\+\_\+\+PLL\+\_\+\+REF\+\_\+\+MIN to FSL\+\_\+\+FEATURE\+\_\+\+MCG\+\_\+\+PLL\+\_\+\+REF\+\_\+\+MAX range.
