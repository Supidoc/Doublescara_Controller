\doxysubsection{Detailed Description}
FTM configuration structure. 

This structure holds the configuration settings for the FTM peripheral. To initialize this structure to reasonable defaults, call the \doxylink{group__ftm_ga165df1038850ac71e511b19fab3e5ab8}{FTM\+\_\+\+Get\+Default\+Config()} function and pass a pointer to the configuration structure instance.

The configuration structure can be made constant so as to reside in flash.
