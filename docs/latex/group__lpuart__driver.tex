Set the LPUART address. 

This function configures the address for LPUART module that works as slave in 9-\/bit data mode. One or two address fields can be configured. When the address field\textquotesingle{}s match enable bit is set, the frame it receices with MSB being 1 is considered as an address frame, otherwise it is considered as data frame. Once the address frame matches one of slave\textquotesingle{}s own addresses, this slave is addressed. This address frame and its following data frames are stored in the receive buffer, otherwise the frames will be discarded. To un-\/address a slave, just send an address frame with unmatched address.

\begin{DoxyNote}{Note}
Any LPUART instance joined in the multi-\/slave system can work as slave. The position of the address mark is the same as the parity bit when parity is enabled for 8 bit and 9 bit data formats.
\end{DoxyNote}
