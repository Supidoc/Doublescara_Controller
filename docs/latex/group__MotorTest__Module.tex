Free\+RTOS task for motor testing and diagnostics. 

This is the main task function for the motor test module. It initializes the stepper motor driver, configures the TMC2209 stepper motor driver, and registers CLI commands for motor control. This task runs indefinitely and should be created as a Free\+RTOS task.
