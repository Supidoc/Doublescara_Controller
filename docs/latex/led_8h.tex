\doxysubsection{Detailed Description}
Simple LED control interface. 

Provides a minimal abstraction for controlling a status LED connected to a GPIO (PTB18 on the target board). The implementation hides the hardware details and returns standard {\ttfamily status\+\_\+t} codes from the SDK.
