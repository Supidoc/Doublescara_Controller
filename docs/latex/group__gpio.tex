The GPIO pin configuration structure. 

Each pin can only be configured as either an output pin or an input pin at a time. If configured as an input pin, leave the output\+Config unused. Note that in some use cases, the corresponding port property should be configured in advance with the PORT\+\_\+\+Set\+Pin\+Config().
