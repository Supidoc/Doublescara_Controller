Initializes the application. 

This function initializes all required modules, peripherals, and tasks for the application. It must be called before running the application.

\begin{DoxyNote}{Note}
This function is typically called once during system startup.

Runs the main application loop. 

This function contains the main application logic and is intended to be called after APP\+\_\+\+Init(). It typically runs indefinitely in an RTOS environment.

\begin{DoxyNote}{Note}
