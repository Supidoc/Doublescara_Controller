Initializes the logging module. 

This function initializes the log message queue, sets up the log file for the current session, and writes the log file header. It must be called before using any other functions in this module.

\begin{DoxyNote}{Note}
Ensure \doxylink{group__DISK__Module_gaa890889c692a0bd26a1bbf2f823c08a7}{DISK\+\_\+init()} is called before initializing the logging module. 

This function is NOT task-\/safe and must be called during system initialization. 

The \doxylink{group__LOG__Module_gaf4aa1ff2f9562bf7fce62e14f5c8df9b}{LOG\+\_\+task()} should be started after this function completes successfully.
\end{DoxyNote}
\begin{DoxyReturn}{Returns}
k\+Status\+\_\+\+Success if the initialization is successful. k\+Status\+\_\+\+Fail if the initialization fails (disk error or queue creation failed).
\end{DoxyReturn}

status\+\_\+t LOG\+\_\+send\+\_\+log\+\_\+message (\begin{DoxyParamCaption}\item[{\doxymbox{\hyperlink{group__LOG__Module_ga6d319de993cc8d4a0b7da064fc94779d}{LOG\+\_\+\+Level\+\_\+t}}}]{level}{, }\item[{char \texorpdfstring{$\ast$}{*}}]{message}{, }\item[{uint8\+\_\+t}]{silent}{}\end{DoxyParamCaption})}



Sends a log message to the log queue. 

This function creates a log message with the specified log level and message content, and adds it to the log queue. The message will be processed and written to the log file by the logging task.


\begin{DoxyParams}[1]{Parameters}
\doxymbox{\texttt{in}}  & {\em level} & The log level of the message. \\
\hline
\doxymbox{\texttt{in}}  & {\em message} & The content of the log message (max 100 characters). \\
\hline
\doxymbox{\texttt{in}}  & {\em silent} & If 1, the message will only be written to the log file. If 0, the message will be output to the console in addition to being logged to the file.\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
This function is TASK-\/\+SAFE and can be called from any task context. 

Messages are queued and processed asynchronously by \doxylink{group__LOG__Module_gaf4aa1ff2f9562bf7fce62e14f5c8df9b}{LOG\+\_\+task()}. 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
If the queue is full, the message will be dropped and the function returns k\+Status\+\_\+\+Fail.
\end{DoxyWarning}
\begin{DoxyReturn}{Returns}
k\+Status\+\_\+\+Success if the message is successfully added to the queue. k\+Status\+\_\+\+Fail if the queue is full or an error occurs.
\end{DoxyReturn}

Task function for processing log messages. 

This function is intended to be run as a Free\+RTOS task. It continuously processes log messages from the queue and writes them to the log file.


\begin{DoxyParams}[1]{Parameters}
\doxymbox{\texttt{in}}  & {\em pv\+Parameters} & Pointer to task parameters (not used in this implementation).\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
This task should be started after \doxylink{group__LOG__Module_gad8fb5a8d2383abe85c0696d80e0372f1}{LOG\+\_\+init()} completes successfully. 

This task is blocked on the message queue and wakes when messages are available. 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
This function does not return; it is expected to run as a perpetual Free\+RTOS task.
\end{DoxyWarning}
