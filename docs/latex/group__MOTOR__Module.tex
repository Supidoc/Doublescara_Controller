\doxysubsection{Detailed Description}
High-\/level API for angle-\/based motor control and safety handling. 

The motor module combines stepper movement control and TMC2209 configuration into one abstraction per motor. All kinematic commands are expressed in mechanical units (degree, degree/\+s, degree/\+s\texorpdfstring{$^\wedge$}{\string^}2, revolutions) and translated internally to step-\/domain operations.
