\doxysubsection{Detailed Description}
Functions for controlling stepper motors using FTM with constant acceleration/\+deceleration. 

This module provides an interface for initializing stepper motor control instances, configuring movement parameters, and executing stepper movements with acceleration and deceleration profiles. It uses Free\+RTOS queues to allow for task-\/safe command submission and asynchronous execution in a dedicated stepper control task.
